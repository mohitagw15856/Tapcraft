# Sounds Directory

Drop your `.wav` or `.mp3` files here and reference them in `config.yaml`.

## Bundled sounds

TapCraft ships with a few placeholder sound names in the default config.
You'll need to add your own sound files — here are some free sources:

- [Freesound.org](https://freesound.org/) — Creative Commons sounds
- [Mixkit](https://mixkit.co/free-sound-effects/) — Free sound effects
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) — Free for commercial use

## Naming convention

Keep filenames simple and descriptive:

```
ping.wav
click.wav
pop.wav
whoosh.wav
error.wav
success.wav
```

## Community sound packs

Want to contribute a themed sound pack? Create a subdirectory:

```
sounds/
├── mechanical/
│   ├── README.md      # Attribution and license info
│   ├── key-press.wav
│   └── key-release.wav
├── nature/
│   ├── README.md
│   ├── raindrop.wav
│   └── wind.wav
└── retro/
    ├── README.md
    ├── 8bit-beep.wav
    └── coin.wav
```

Include a `README.md` in your pack with proper attribution!
